# 资料收集模板（Module A）

> **核心原则**：多维度、多来源、多视角收集信息，确保 BP 核心论点均有数据支撑。

---

## A1. 市场机会调研

**web_search 查询**（分层搜索策略）：
```
- "{细分领域} 市场容量 潜在规模"
- "{行业赛道} 未来三年增长趋势 预测"
- "{行业} 行业白皮书 灼识咨询 中商产业研究院 弗若斯特沙利文"
- "{客户群体} 核心诉求 业务挑战"
- "{行业} 采纳率 市场饱和度 成长空间"
- "{行业} total addressable market sizing"
```

**fetch 深度分析**：
```
fetch(
  url: "https://industry-research.example.com/market-outlook-2025.pdf",
  prompt: "汇总该领域的整体市场容量、年复合增长率、各层级市场规模（TAM/SAM/SOM）、细分赛道分布、核心增长驱动力"
)
```

**数据多重校验**：
- 市场规模数据需来自至少 2 个独立信源（仅 1 个来源时需继续检索，不可采用）
- 不同来源数据差异超过 25% 时需说明统计口径差异
- 行业痛点量化指标（如人工耗时、错误率等）需注明研究机构或报告出处

**SAM/SOM 自洽性校验（MUST）**：
- 完整校验规则见 [constraints.md](constraints.md) 中的「数据一致性红线」
- 示例：产品定价 5-20 万/年，目标客户 8000 家 → SAM = 4-16 亿/年（避免使用"年均 IT 预算 60-120 万"得出 480-960 亿）

**写入草稿文档**（使用 `dws doc update --node <DRAFT_DOC_URL> --markdown <内容> --mode append --format json`）：
```markdown
## 市场机会调研

### 市场容量测算
| 维度 | 金额 | 数据来源 | 统计年度 |
|------|------|---------|---------|
| TAM（潜在总市场） | 3,200 亿元 | 灼识咨询 | 2024 |
| SAM（可服务市场） | 480 亿元 | 弗若斯特沙利文 | 2024 |
| SOM（可获得市场） | 35 亿元 | 团队测算 | 2026E |

### 增长趋势分析
- 年复合增长率：22.8%（中商产业研究院，2025-2030 预测）
- 核心驱动：政策红利释放、技术迭代加速、市场需求激增
- 市场渗透度：当前约 8%，预期 2028 年达到 42%

### 客户核心痛点
| 痛点类别 | 影响程度 | 现行方案 | 存在缺陷 |
|---------|---------|---------|---------|
| 流程效率低下 | 高 | 传统信息化系统 | 定制开发周期长、维护成本高 |
| 信息分散割裂 | 高 | 人工汇总整理 | 数据时效性差、准确性不足 |
| 决策依据薄弱 | 中 | 基础报表工具 | 使用门槛高、应用场景受限 |
```

## A2. 竞争格局调研

**web_search 查询**：
```
- "{行业} 竞争对手分析 市场地位"
- "{竞品A} 功能特点 产品优势"
- "{行业} 新兴企业 融资动态"
- "{竞品A} 商业模式 盈利模式"
- "{行业} 竞品对比 评测"
- "{竞品A} 客户评价 口碑"
```

**fetch 深度调研**：
```
fetch(
  url: "https://competitor-analysis.com/product-overview",
  prompt: "提取该企业的产品定位、核心能力、价格体系、典型客户、融资轮次与金额"
)
```

## A3. 内部资料收集

**dws doc search 查询**：
```bash
dws doc search --query "产品说明" --format json
dws doc search --query "盈利模式" --format json
dws doc search --query "融资历程" --format json
dws doc search --query "核心团队" --format json
```

**web_search 补充查询**：
```
- "{公司名} 主要产品 功能特性"
- "{公司名} 成功案例 客户反馈"
- "{公司名} 创始人团队 履历"
- "{公司名} 经营数据 关键指标"
```

**写入草稿文档**（使用 `dws doc update --node <DRAFT_DOC_URL> --markdown <内容> --mode append --format json`）：


```
- "{行业} 上市公司 市销率 估值倍数"
```

```
fetch(
  url: "https://benchmark-report.com/saas-metrics-2024.pdf",
```

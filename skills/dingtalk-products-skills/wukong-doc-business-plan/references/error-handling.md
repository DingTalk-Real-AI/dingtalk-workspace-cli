# 错误处理与异常应对

---

## 工具调用错误

| 错误 | 原因 | 处理方式 |
|------|------|----------|
| `dws doc create` 失败 | 认证过期或权限不足 | 请重新执行上一条命令（最多重试两次） |
| `dws doc update` 报 `invalidRequest.inputArgs.invalid` | `--node` 使用了裸 `nodeId` 而非完整文档 URL | 必须使用 `dws doc create` 返回的 `docUrl` 字段（如 `https://alidocs.dingtalk.com/i/nodes/xxx`）作为 `--node` 参数 |
| `dws doc update` 报 `invalidRequest.inputArgs.invalid`（内容相关） | `--markdown` 内容过长（超过约 2000 字符）或含特殊字符 | 分块写入：每次 `--mode append` 控制在 1500 字符以内，逐章追加 |
| `dws doc update` 内容格式错乱 | `--markdown` 参数中使用了字面量 `\n` 而非真实换行符 | 确保 Markdown 内容使用真实换行符（Unicode U+000A） |
| `dws doc search` 无结果 | 内部无相关文档或关键词不匹配 | 尝试更宽泛的关键词，或跳过内部资料收集步骤 |
| Writer Agent 生成内容过长 | 单次 `--markdown` 内容超出限制 | 先 `dws doc create` 创建空文档，再分章节使用 `dws doc update --mode append` 逐步追加 |

## Shell 命令错误

| 错误 | 原因 | 处理方式 |
|------|------|----------|
| shell 命令报 `No such file or directory` | 文件名含空格未加引号，被 shell 拆分为多个参数 | 文件名含空格时必须用双引号包裹，如 `cat "BI 分析 A 轮融资计划.md"` |

## 网络与搜索错误

| 错误 | 原因 | 处理方式 |
|------|------|----------|
| `web_search` 无结果 | 搜索关键词过于具体或行业过于小众 | 扩大搜索范围，尝试英文关键词，或使用 `fetch` 直接访问已知报告 URL |
| `web_search` 超时 | 网络连接不稳定 | 重试 1-2 次，若仍失败则跳过该搜索项并标注"需补充资料" |
| `fetch` 返回空内容或 403 | 目标网站反爬或需要登录 | 尝试其他来源，或提示用户手动提供该资料 |

## 数据质量问题

| 错误 | 原因 | 处理方式 |
|------|------|----------|
| 市场数据来源不足 | 行业过于细分，公开数据有限 | 标注"需补充资料"，建议用户提供内部数据或委托第三方调研 |
| 市场规模数据差异 > 30% | 不同来源口径不一致 | 注明各来源的口径差异（如含税/不含税、全球/中国），取中间值或保守值 |
| 竞品信息不完整 | 竞品为非上市公司，公开信息有限 | 标注已知信息，缺失项标注"需补充"，建议用户通过行业渠道获取 |

## 服务不可用

| 错误 | 原因 | 处理方式 |
|------|------|----------|
| `dws` 服务整体不可用 | 钉钉文档服务故障或维护 | 提示用户稍后重试，或先以对话形式输出 BP 框架供用户参考 |
| Agent 调用超时 | Writer Agent 处理时间过长 | 减少单次传入的内容量，分更小的块写入 |

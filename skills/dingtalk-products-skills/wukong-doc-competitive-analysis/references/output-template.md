# 竞品分析报告输出模板

> 本模板基于业界主流竞品分析框架（参考 HubSpot、Asana、Klue 等）设计，适配钉钉文档输出格式。

## 文档命名规范

```
竞品研究：{目标产品} 竞争格局分析 {YYYY-MM}
```

---

## 报告结构总览

```
封面信息
├── 0. 执行摘要（决策层速读）
├── 1. 竞争格局概览（竞品分类 + 定位象限）
├── 2. 竞品深度画像（每个竞品独立分析）
├── 3. 横向能力对比（功能矩阵 + 体验评分）
├── 4. 商业模式解析（定价 + 增长飞轮）
├── 5. 市场与用户洞察（规模 + 口碑）
├── 6. 战略机会与行动计划
└── 附录（数据来源 + 原始评价）
```

## 写作规范

| 规范项 | 要求 |
|-------|------|
| 数据来源 | 所有数据点必须标注来源 URL 和获取日期 |
| 定价数据 | 只能来自官网直采，不得引用博客、评测或第三方摘要 |
| 缺失数据 | 填写「需补充」，禁止估算或编造 |
| 功能状态 | 「🔶 部分支持」必须在表格下方补充具体限制说明 |
| 行动建议 | 每条必须包含：具体行动 + 可衡量目标 + 时间窗口 |
| 口碑评分 | 必须注明评分平台和评分数量（如：4.3/5，基于 120 条评价） |
